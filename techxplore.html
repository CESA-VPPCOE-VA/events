Winner of TechXplore


Event Name: TechXplore

10/07/2024 Rizvi Ahmed Abbas

Venue: Computer Labs 3rd Floor
Total participants : 80
Faculty Coordinator : Prof. Shefali Raina

Objective of the event:

The primary goal of this event is to educate the participants on C and JAVA programming language.

Outcome of the event:

The outcome of the TECHXPLORE event is to improve the coding skills of participants and to enhance their problems solving ability and speed, and also it provides practical learning experiences.

The event had 3 rounds, where the 1st round code was based on C language, 2nd round code on JAVA and 3rd round on CORE JAVA programming language. To manage the participants efficiently, we divided teams into 3 batches for 1st round, one for 2nd and 3rd.We called all students batch wise and well organized round 1 with discipline. From each batch 2 teams went to next round.

Initially, the snippets were given to each team, they had to solve that code and get the output of three digit number which will unlock the number lock, these was common for all 3 rounds. The Rules were explained properly to the participated teams. And we also provided them rough sheets to solve the code.

We also invited our respected principal Dr. Alam N Shaikh, vice principle Dr. Pravin Yadav & HOD of Computer department Dr. Rais Mulla to attend this event. All of them appreciated our event.

We provided participation certificates to all the participants and the winners were awarded with trophy and certificates.
