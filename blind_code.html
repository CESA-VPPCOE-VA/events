Blind Coding

Event Name: BLIND CODING

25/08/2023 Ekta Champaneri

Venue: Computer Labs 3rd Floor
Total number of participants : 60 students
Faculty Coordinator : Prof. Shefali Raina

Objective of the event:

The primary goal of this event is to educate the participants on c programming language.

Outcome of the event:

The outcome of the blind coding event is to improve the participants’ coding skills, the enhancement of typing accuracy and speed, the provision of practical learning experiences.

The technical event “Blind Coding” was organized by CESA (Computer Engineering Students Association) in association with CSI (Computer Society of India) on August 25,2023. Students from all the department participated in the event.

The event started at 11:00 am. The participants showed great interest in the event as more than 100 responses were received. This speaks to the popularity and appeal of the blind coding competition.

To manage the participants efficiently, we divided them into 5 batches, each consisting of 20 participants, while the other participants waited in classroom 314 till the earlier participants finished their task. This allowed for a manageable number of participants per batch, making it easier to coordinate and oversee the competition.

Allocating two labs for the event ensured that there was enough space for all participants to comfortably take part in the competition.

Initially, the code was written on the board for this task, rules were explained, and then participants were asked to open Notepad or VS Code Software on their computers, whichever they were most comfortable with. They then had to set their cursors, turn off the PC, and begin typing the code. Within 10 minutes, they were expected to finish the task.

The winners were determined based on three main criteria:

  • Time: The speed at which participants could accurately type the code.
  • No of Errors: The accuracy of the code typed, minimizing syntax and logical errors.
  • Execution: The successful compilation and execution of the code typed.

After evaluating all participants in each batch based on the specified criteria, we shortlisted the top three performers. In this way the event was successfully conducted.

The winners of the blind coding were :-

  1. Nikhil Damre (S.E B Comps)
  2. Sanil Jadhav (T.E B Comps)
  3. Krish Patel (F.E IT)
